CM4-IO-BASE-A + USB HDMIアダプター Raspberry Piコンピュートモジュール4用 FFC経由のUSBおよびHDMIコネクター

概要

  • これはRaspberry Piコンピューティングモジュール第4世代用の基本のIOボードです。 CM4-IO-BASE-Aは、Raspberry Piコンピュートモジュール4で使用できるベースボードです。 電源には5V/2.5A USB Type Cインターフェースを使用しています。
  • CM4-IOベース専用USB HDMIアダプタ、FFCコネクタを標準コネクタに変換。

特徴

  • Raspberry Pi CM4の評価や、エンド製品に統合されているのに適しています。 クレジットカードサイズのボディにパワフルな機能。
  • 標準CM4ソケットと色分けされたRaspberry Pi 40ピンGPIOヘッダー。 コンピューティングモジュール4 Lite/EMMCシリーズモジュールに適しています。
  • オンボード複数のコネクタを含む:CSI/DSI/FAN/HDMI/USB/RJ45ギガビットイーサネット/マイクロSDカードスロット/M.2スロット。

パッケージ内容

  • CM4-IO-BASE-A x1。
  • USB HDMIアダプター x 1。
  • FFCケーブル 6ピン 1.0 ピッチ 60mm x1
  • FFCケーブル 20ピン 0.5ピッチ 60mm x 1
  • USB-A - USB-Cケーブル 1m x 1。
  • SSD取り付けネジx1。
